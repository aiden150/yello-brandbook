# Brandbook Directions

Use this skill when the user wants a brandbook, brand direction board, or identity system built from a brief.

## Workflow

1. Start with exactly 3 creative directions.
2. Make each direction meaningfully different in emotional tone, visual weight, and message emphasis.
3. Map user-provided reference images to those three directions.
4. Mark one direction as selected.
5. Build the final brandbook around the selected route with:
   - brand promise
   - voice and tone
   - color system
   - typography direction
   - usage principles
   - product narrative

## Output rules

- Show the 3 directions before the selected system.
- Keep the selected route tied back to the original product strategy.
- Translate strategy into a visual system, not just a mood board.
- Reuse the same structure across future brandbooks so the workflow stays consistent.

## Good defaults

- For investment or B2B platform brands, favor clear hierarchy, restrained confidence, and data-forward language.
- Use short slogan lines or route names for each direction.
- Prefer one selected route with a compact rationale instead of trying to combine all three.
